vue基础

发表于 2021-11-14 分类于 VUE

Vue基础概念:

1.什么是vue:

它是一个渐进式的的框架,也可以把它当做是一个库, 渐进式: 便于复用

我们不需要自己去配置webpack,他会自动的帮我们配置好

2.脚手架:

脚手架有什么好处?

​ 它是不用任何的配置,马上就能使用,快速帮助我们搭载项目开发环境

如何安装脚手架:

​ 1.先打开终端书写 vue-V(大写)查看有没有安装vue ,如果没有先下载

​ 2.创建脚手架的指令: yarn global add @vue/cli

​ 3.vue create 项目名字(不能是汉字)

​ 4,yarn serve 启动项目

了解脚手架里面的主要文件

​ node_modules - 都是下载的包

​ public/index.html - 浏览器运行的网页

​ src/main.js - webpack打包的入口

​ src/App.vue - Vue页面入口

​ package.json - 项目配置信息

main.js和App.index.html关系

​ APP.vue=>main.js=>index.html

脚手架配置文件:

​ vue.config.js里面去配置,(配置完要重启服务器)

什么是eslint:

它是一个代码的检查工具,里面可以配置一些规则,如果违反了他就会报错

怎么关闭?

​ 在vue.config.js里面设置lintOnSave为false 重启服务器就行

单个vue文件
好处:

​ 是一个独立的作用域,防止了变量污染问题

​ 注意事项:

​ template里面只能有一个跟标签并且写的所有内容都要包裹在这个根标签里面

​

3.Vue指令

什么是插值表达式:

​ 双大括号包裹的的vue变量放在标签里面用 不能写语句例如:if() while()

​ vue变量声明:

必须要声明在data函数内,return的对象里面,键值对的方式

MVVM是什么

这个一种设计模式 当视图改变了相对应的数据也改变了,数据改变了视图也改变了,我们也可以理解为是一种双向的绑定

好处:

减少了DOM操作,提高开发效率

v-bind:

标签设置为动态属性,给dom标签设置, 写在标签里面,

  • 语法：v-bind:属性名="vue变量"
  • 简写：:属性名="vue变量"
事件绑定(v-on);

必须写在和data同级的methods里面

  • 语法
    • v-on:事件名=”methods中的函数”
    • v-on:事件名=”methods中的函数(实参)”

  • 简写: @事件名=”methods中的函数”

修饰符:

@事件名.修饰符=”methods里函数”

  • .stop - 阻止事件冒泡
  • .prevent - 阻止默认行为
  • @keyup.enter - 监测回车按键
  • @keyup.esc - 监测返回按键
v-model:
  • 语法: v-model=”vue数据变量” 现在用于表单
  • 双向数据绑定
    • 数据变化 -> 视图自动同步
    • 视图变化 -> 数据自动同步

非数组 :关联的是复选框的checked属性
数组 :关联的是复选框的value属性

下拉菜单:关联在select上

v-model修饰符

v-model.number 转为数字
v-model.trim 去掉字符两边的空格
v-model.lazy 失去焦点时赋值

v-text和v-html

v-text 只传一个字符串不会解析
v-html 把字符串里的标签解析出来
(v-html 和v-text )都会覆盖里面的值

v-show和v-if

v-show 显示或隐藏 采用display:none 频繁切换 true显示 false隐藏
v-if=”变量” 显示或隐藏 采用从DOM树直接移除 条件达到后显示
v-else 与之相反

v-for:

v-for=(值变量,索引变量) in 目标结构
v-for=”值变量 in 目标结构” 不要索引这样写
v-for=(值,属性) in 对象 可以遍历对象
